🚀 Built an Interactive Drum Kit using HTML, CSS & JavaScript

Excited to share a hands-on project where I developed a browser-based drum kit that responds to both mouse clicks and keyboard inputs in real time.

🔹 Key Highlights:
• Event-driven architecture using JavaScript
• Integrated keyboard & click event listeners for seamless interaction
• Dynamic audio playback mapped to specific keys
• Clean UI with responsive feedback animations

🔹 Tech Stack:
HTML | CSS | JavaScript

This project strengthened my understanding of DOM manipulation, event handling, and user interaction design.

Looking forward to building more interactive and scalable web applications.
